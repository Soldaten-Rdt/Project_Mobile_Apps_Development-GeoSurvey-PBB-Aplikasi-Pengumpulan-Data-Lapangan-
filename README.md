# Project_Mobile_Apps_Development-GeoSurvey-PBB-Aplikasi-Pengumpulan-Data-Lapangan-
Dokumentasi dari project Mobile Apps Development.
Membuat aplikasi berbasis web yang berfunsgi untuk membantu tugas para Enumerator dalam mengerjakan tugasnya ketika melakukan Geosurvey dengan tujuan untuk membantu perhitungan pajak PBB
Struktur Aplikasi :

a. Platform = Flutter

b. Bahasa Pemrograman = DART dan SQL

c. Database = SQLite dan MYSQL
